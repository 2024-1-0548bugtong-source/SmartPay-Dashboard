## 🚀 Daily Startup (5 minutes)

1. **Power On the Machine**
   - Turn on the SmartPay Arduino device
   - Wait for LCD display to show "SmartPay Ready"

2. **Open Dashboard**
   - Open web browser (Chrome, Edge, Firefox, Safari)
   - Go to your dashboard URL (bookmark it!)

3. **Connect Arduino**
   - Click "Connect Arduino" button
   - **If USB:** Select the COM port from the dropdown
   - **If Bluetooth:** Use your phone's Bluetooth settings (PIN: 1234)

4. **Verify Status**
   - You should see "SmartPay Ready" message
   - LCD display shows: "SmartPay Ready"
   - Device is ready to accept customers!

---

## 💰 Selling Flow

1. **Customer Places Product** → Scale detects weight
2. **Display Shows Price** → e.g., "Pay Product One (PHP5)"
3. **Customer Inserts Coins** → Scale counts total amount
4. **Display Shows Remaining** → e.g., "Remaining: PHP2"
5. **Payment Complete!** → Buzzer sounds, "Payment OK"
6. **Dispense Product** → Device ready for next customer

## ⚠️ Troubleshooting

| Problem | Solution |
|---------|----------|
| **Dashboard won't load** | Check internet, refresh page, wait 30 seconds |
| **"Connect Arduino" button not working** | Restart browser, check USB cable, restart Arduino |
| **Scale not detecting products** | Ensure scale is calibrated, check nothing blocking sensor |
| **No coins detected** | Verify coin weights match calibration, clean sensors |
| **Bluetooth not connecting** | Unpair and re-pair HC-05, check PIN is 1234 |
